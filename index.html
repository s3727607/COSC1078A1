Project Idea

· Overview

VR helmet has always been a wearable device that I want to have. It can completely bring users into a virtual world, and use bioelectricity to stimulate the brain to bring people close to the real feeling. This idea is from an anime "Sword Art Online" , this anime is about what happened in this VR helmet game. The closest thing to this project is VR glasses, but VR glasses are not able to give people a more exciting feeling now.

· Motivation

In the current era, most real-world technologies are no longer able to satisfy the users of the game, and the complete virtual world gives a lot of embarrassment. If people can do something in a completely virtual world, this is a question that every big player will think about, which is why I think of this project. Augmented reality, holographic images, these technologies are all about VR, until Google glasses announced the end of production in 2015, the topic is gradually less. But people who are really interested in VR are still studying, such as Google, Microsoft, Samsung, they are still committed to the development of VR glasses, but this is only a visually new experience. And what I want is a wearable device that can be feedback from the sense of touch and vision. This requires bioelectronics technology on the product.

· Description

This VR Helmet's concept features a head-mounted display (HMD) and a game engine (Unity3D)-based gesture recognition technology, as well as bioelectronics technology for safety.

The workflow of this product is to receive user input from the gesture recognition device in the virtual reality operating system to simulate the mouse and keyboard, and provide output through the head-mounted display and bioelectric signal transmitter & bio-probe to achieve user's comprehensive immersive experience.

This will bring a new boom in the hardware market, which can show users an experience that they have never had before. For example, the wearable device can be fully fed back to the user's palm touch, which can limit the player's action by changing its hardness to virtualize the existence of the object in the program. And by radiating micro-current to the player's hand, it brings almost real touch, which can deepen the user's impression of the game and a more realistic operating experience. There is also a wide-angle 156° curved display, which is ergonomically perfect for Monocular horizontal perspective and gives the picture a more three-dimensional look.

· Tools & Technologies

Hardware: Hardware requires a head-mounted display, a pair of somatosensory gloves (including multiple sets of electrode patches and multiple bionic joint sets), a set of built-in stereos, and a small vibration motor (simulating the vibration of the head)

Software:

1. Write code directly using android opengl api, and also provide some useful VR components. But the downside is that you need to be familiar with open gl.

2. Provides the interface for ios open gl programming, same as above.

3. Provide the unity plugin, you can import it in unity, then use c# or js for development, and get started quickly. In addition, Cardboard VR (and Gearvr) is supported by default on the latest unity.

4. Provide native support for unreal.
